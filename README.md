# Fungal Guessing Game
This is a C++ program containing data about six edible fungi:
* Chanterelle (_Cantharellus cibarius_)
* Oyster mushroom (_Pleurotus ostreatus_)
* Morchelle (_Morchella esculenta_)
* Indigo milk cap (_Lactarius indigo_)
* Ringless honey mushroom (_Desarmillaria tabescens_)
* Bolete (_Boletus edulis_)

The source code can be found in `main.cpp`.

## To play the game, download and run `FungalGuessingGame.exe`, then select Option 1. To view a detailed list of fungi, select Option 2.
### How to play:
Mentally choose one fungus. The program will ask you ten questions about a hypothetical fungus; select the answers that most closely match your choice.
#### (Note that typing random numbers will not necessarily give you an accurate result!)
